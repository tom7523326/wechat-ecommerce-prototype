# 微信电商能力提升产品 - 逻辑重构方案

基于原思维导图，重新梳理清晰的产品逻辑和页面流程

---

## 一、核心业务流程

### 用户旅程（User Journey）

```
1. 学员准入
   ↓
2. 完成基础评估
   ↓
3. 系统生成：数字化名片 + 能力模型
   ↓
4. 分配等级（C/B/A/S）
   ↓
5. 推荐学习路径
   ↓
6. 学习课程 + 参与活动
   ↓
7. 能力提升 → 等级晋升
   ↓
8. 持续学习 + 社区交流
```

---

## 二、清晰的信息架构

### 主导航结构（5个一级入口）

```
🏠 首页           - 数据概览、学习进度、待办任务
📚 学习中心       - 课程、知识图谱、学习路径
👤 我的名片       - 数字化名片、能力模型、资源对接
⭐ 学员等级       - SABC分层、权益、晋升
💬 社区活动       - 社区交流、运营活动
```

## 三、页面跳转逻辑

### 首页跳转关系
```
首页
├─ 粉丝数卡片 → 数据统计页
├─ GMV卡片 → 数据统计页
├─ 课程完成卡片 → 学习中心
├─ 学员等级卡片 → 学员等级页
├─ "继续学习"按钮 → 课程中心
├─ 课程卡片 → 课程详情
├─ 能力雷达图点击 → 知识图谱
├─ 动态点击 → 动态详情
├─ 消息按钮 → 消息中心
└─ 头像 → 个人中心
```

### 学习中心跳转关系
```
课程中心
├─ 课程卡片 → 课程详情
├─ "开始学习" → 课程详情
└─ 筛选标签 → 筛选结果

知识图谱
├─ 节点点击 → 右侧详情面板
├─ "开始学习" → 课程中心（对应课程）
└─ 关联课程 → 课程详情

学习路径
├─ 路径步骤 → 对应课程
├─ "继续学习" → 当前课程
└─ 里程碑奖励 → 等级页面
```

### 我的名片跳转关系
```
数字化名片
├─ "编辑"按钮 → 编辑模式
├─ "分享"按钮 → 分享弹窗
├─ 能力雷达图 → 知识图谱
├─ 服务项目 → 资源对接
└─ 需求项目 → 资源对接
```

### 学员等级跳转关系
```
学员等级页
├─ 等级卡片点击 → 该等级详情
├─ "申请晋级"按钮 → 晋级申请表单
└─ 权益详情 → 对应功能页面
```

### 社区活动跳转关系
```
社区首页
├─ "发布"按钮 → 发布弹窗/页面
├─ 动态卡片 → 动态详情
└─ 用户头像 → 该用户名片

活动中心
├─ 活动卡片 → 活动详情
└─ "报名"按钮 → 报名表单

动态详情
├─ 点赞、评论、转发
└─ 话题标签 → 话题页面
```
