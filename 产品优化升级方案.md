# 微信电商能力提升平台 - 产品优化升级方案

## 一、产品定位升级

### 1.1 核心价值主张
从"能力提升"升级为"**全周期数字化经营生态赋能**"
- 精准筛选 → 阶梯培育 → 持续赋能 → 数据追踪
- 从单点培训到生态共建
- 从能力提升到产业升级

### 1.2 产品愿景
构建乡村数字化经营生态，通过SABC分层体系，实现：
- 学员增收
- 乡村产业升级
- 标杆案例复制
- 成效反哺培训体系优化

---

## 二、信息架构重构

### 2.1 核心模块升级

#### 原有模块 → 升级后模块

1. **学员准入** → **精准筛选系统**
   - 基础信息采集
   - 数字化基础评估
   - 地域产业匹配
   - 数字化需求分析
   - 学员准入标准

2. **知识图谱** → **阶梯培育体系**
   - C基础入门层（300人）
   - C-B能力提升层（100人）
   - B-A标杆储备层（30-50人）
   - S生态共建层（5-10人）

3. **学员分层** → **SABC分层体系**
   - 成长数据（愿力）
   - 经营数据（能力）
   - 积分体系（激励）
   - 权益体系（赋能）

4. **学习路径** → **全周期培育路径**
   - 培训前：招募与需求匹配
   - 培训中：线上培训+线下培训+活动验证
   - 培训后：持续赋能（共创社群）

5. **活动中心** → **社群运营中心**
   - 日常运营
   - 主题陪跑营
   - 优秀案例分享
   - 资源对接活动
   - 政企联合活动

6. **新增：数据中台**
   - 全周期数据采集
   - 可视化看板
   - 成效量化
   - 产业提升对比

7. **新增：资源对接中心**
   - 产品库
   - 人才库
   - 货盘库
   - 需求池

## 三、设计规范升级

### 3.1 视觉设计系统

#### 色彩体系
- **主色**：微信绿 #07C160（品牌一致性）
- **辅助色**：
  - S级：深绿 #00A86B（生态共建）
  - A级：蓝色 #1890FF（标杆储备）
  - B级：橙色 #FA8C16（能力提升）
  - C级：浅橙 #FFA940（基础入门）
- **功能色**：
  - 成功：#52C41A
  - 警告：#FAAD14
  - 错误：#FF4D4F
  - 信息：#1890FF

#### 字体系统
- **主字体**：PingFang SC / -apple-system
- **字号层级**：
  - H1：32px（页面标题）
  - H2：24px（区块标题）
  - H3：20px（卡片标题）
  - Body：16px（正文）
  - Small：14px（辅助信息）
  - Caption：12px（说明文字）

#### 间距系统
- **基础单位**：4px
- **间距层级**：4px, 8px, 12px, 16px, 24px, 32px, 48px

#### 圆角系统
- **小圆角**：4px（按钮、标签）
- **中圆角**：8px（卡片）
- **大圆角**：12px（大卡片、弹窗）

#### 阴影系统
- **Level 1**：0 2px 8px rgba(0,0,0,0.08)（卡片）
- **Level 2**：0 4px 16px rgba(0,0,0,0.12)（悬浮）
- **Level 3**：0 8px 24px rgba(0,0,0,0.16)（弹窗）

### 3.2 组件设计规范

#### 按钮系统
- **主要按钮**：高度44px，圆角4px，16px内边距
- **次要按钮**：高度40px，边框样式
- **文字按钮**：高度36px，无背景
- **图标按钮**：40x40px，圆形

#### 卡片系统
- **标准卡片**：16px内边距，8px圆角，Level 1阴影
- **悬浮卡片**：hover时Level 2阴影，轻微上移
- **信息卡片**：24px内边距，12px圆角

#### 表单系统
- **输入框**：高度44px，16px内边距，2px边框
- **标签**：14px字号，8px下边距
- **错误提示**：12px字号，红色，4px上边距
