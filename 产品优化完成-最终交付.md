## 核心创新点

### 1. 成长值系统
**突破传统单一GMV考核**

计算公式：
```
成长值 = 经营数据得分（50%）
       + 学习成长得分（30%）
       + 社群贡献得分（20%）
```

6种获取途径：
- 📊 经营数据提升：最多+100/周
- 📚 完成学习任务：+10~50/次
- 🎯 参与社群活动：+5~30/次
- 💡 分享成功案例：+30~100/次
- 🤝 资源对接成功：+50/次
- 👨‍🏫 帮助其他学员：+10~30/次

### 2. 可视化金字塔
**创新的SABC展示方式**

- 使用clip-path实现梯形金字塔
- 分层配色（金/橙/蓝/灰）
- pulse动画标注当前位置
- 点击查看等级详情

### 3. 数据驾驶舱
**一屏掌握全局**

- 4大核心数据卡片（可点击跳转）
- 智能推荐（AI驱动）
- 待办任务（任务驱动成长）
- 实时数据更新

### 4. CSS变量系统
**便于主题切换和维护**

所有颜色、间距、圆角使用CSS变量定义：
```css
:root {
    --primary-color: #07C160;
    --level-s: #FFB800;
    --spacing-md: 16px;
    ...
}
```

### 5. 底部Tab导航
**移动优先策略**

- 5个核心功能入口
- 自动高亮当前页面
- 固定底部，便于操作
